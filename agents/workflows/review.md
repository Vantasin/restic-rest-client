# Review Workflow Index

Use the specific review workflow that matches the layer you changed.

## Review Types

- [`agent-review.md`](./agent-review.md): run after core changes to `AGENTS.md`,
  `agents/`, or the repo's agent guidance structure
- [`repo-review.md`](./repo-review.md): run after core changes to repository
  behavior, scripts, templates, scheduling, backup scope, or other functional
  behavior

## Logging

If either review causes follow-up edits, log those edits with
[`change-logging.md`](./change-logging.md).
